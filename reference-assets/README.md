# 参考资料文件夹

这个文件夹用于存放参考图片和音效,帮助优化生成器。

## 📁 文件夹结构

### `/images` - 参考图片
请在这里放入你喜欢的像素怪物图片:
- 推荐 3-5 张图片
- 支持格式: PNG, JPG, GIF
- 建议像素风格的图片
- 图片会作为设计参考,不会直接使用

**示例:**
- `monster1.png` - 可爱风格
- `monster2.png` - 恐怖风格
- `monster3.png` - 机械风格
- 等等...

### `/audio` - 参考音效
请在这里放入你喜欢的怪物音效:
- 推荐 2-3 个音效
- 支持格式: WAV, MP3, OGG
- 建议 1-3 秒的短音效
- 音效会作为合成参考

**示例:**
- `creature_roar.wav` - 咆哮声
- `cute_squeak.wav` - 可爱叫声
- `robotic_beep.wav` - 机械声
- 等等...

## 🎨 使用方法

1. 把参考文件放入对应文件夹
2. 告诉 AI 你放好了
3. AI 会分析这些文件并优化生成算法
4. 重新构建项目即可看到改进效果

## 📝 注意事项

- 这些文件**不会**被打包到最终网站中
- 仅用作设计参考和算法优化
- 可以随时添加或更换参考文件
